## About

Playyyy is a curated collection of over 100 high-quality browser-based games. Built as a modern web portal, it offers a clean, responsive, and visually appealing interface with a glassy dark theme.
All games are playable directly in the browser without downloads or emulators. The platform uses a dynamic JSON-powered system, making it extremely easy to add new games.
Whether you're looking for classics like 2048, Slope, Subway Surfers, Friday Night Funkin', or modern hits like Drift Hunters and Territorial.io — Playyyy brings them all together in one elegant place.

## Features

- ✅ Modern Glassy Dark Design – Beautiful, futuristic look with blur effects
- ✅ Easy to Maintain – Add new games by simply updating the JSON
- ✅ Responsive Grid – Works perfectly on mobile, tablet, and desktop
- ✅ Dedicated Game Page – Clean iframe player with description and back button
- ✅ Fast & Lightweight – No heavy frameworks, pure vanilla code
- ✅ Full-screen Ready – All games support fullscreen mode
